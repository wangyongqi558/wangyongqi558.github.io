### 1. Abstract
An end-to-end framework that eliminates the dependency on pre-trained trajectory detectors, allowing for a more seamless open-vocabulary detection.

### 2. Methodology

![Methodology](/images/tpami_method.png)
* **Relationship-Aware Trajectory Detection**: Integrating interaction context into object queries.
* **CLIP Distillation**: Mapping frame-level features to CLIP latent space.

### 3. Key Results
Robust performance across different datasets (VidVRD to VidOR).
